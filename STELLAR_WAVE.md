# Axiom State Monitor — Stellar Drips Wave 2026

## What is Drips Wave?

Drips Wave is the 2026 Stellar contributor program operating on a **Fix, Merge, Earn** model. Contributors pick up tagged issues, submit PRs, and earn XLM rewards upon merge.

## How to Participate

1. Browse open issues tagged `[WAVE-TRIVIAL]`, `[WAVE-MEDIUM]`, or `[WAVE-HIGH]` in `specs/tasks.md` and the GitHub Issues tab.
2. Comment on the issue to claim it — first claim wins.
3. Fork the repo, implement the fix on a branch named `wave/<WAVE-ID>-short-description`.
4. Open a PR using the Wave PR template (`.github/ISSUE_TEMPLATE/wave_pr.md`).
5. Pass CI (build + clippy + tests) and receive maintainer review.
6. Merge → Earn. Rewards are distributed within 7 days of merge.

## Reward Tiers

| Tag | Reward (XLM) |
|-----|-------------|
| WAVE-TRIVIAL | 10 XLM |
| WAVE-MEDIUM | 50 XLM |
| WAVE-HIGH | 150 XLM |

## Rules

- One active claim per contributor at a time.
- Claims expire after 72 hours of inactivity.
- PRs must include tests for WAVE-MEDIUM and WAVE-HIGH issues.
- No AI-generated code submissions without disclosure.

## Maintainers

Drips Wave is maintained by the Drips Wave core team. Reach out via GitHub Discussions for questions.
